## Industrial IoT (IIoT) Platforms
*Middleware for connecting factory machines to the cloud.*

* [EdgeX Foundry](https://www.edgexfoundry.org/) - A vendor-neutral open source project building a common open framework for IoT edge computing.
* [ThingsBoard](https://thingsboard.io/) - Open-source IoT platform for device management, data collection, processing, and visualization.

## Open Source PLCs & SCADA
*Software for controlling industrial machinery.*

* [OpenPLC](https://openplcproject.com/) - The first fully open-source Programmable Logic Controller that meets the IEC 61131-3 standard.
* [ScadaBR](http://www.scadabr.org.br/) - Open source Supervisory Control and Data Acquisition system.

## Edge Computing & Protocols
*Communication standards for the factory floor.*

* [MQTT](https://mqtt.org/) - The standard for IoT messaging.
* [Node-RED](https://nodered.org/) - A flow-based programming tool for wiring together hardware devices, APIs, and online services.
